================================================================================
1. PROJECT OVERVIEW
================================================================================
WorkBoard is a premium, high-performance full-stack web application designed for 
efficient team collaboration and project management. It features a stunning 
dark glassmorphism UI with role-based access control, real-time-like 
interactions, and a powerful Kanban board.

================================================================================
2. KEY FEATURES
================================================================================
- 🔐 AUTHENTICATION: Secure JWT-based login and signup with role selection.
- 📊 DASHBOARD: Comprehensive overview of task statistics, overdue items, 
    and recent team activity.
- 📁 PROJECT MANAGEMENT: Create and manage projects with custom color coding 
    and team assignment.
- ✅ TASK TRACKING: Full Kanban board with drag-to-update status, priority 
    labels, and due date tracking.
- 👥 TEAM COLLABORATION: Add members to projects and assign tasks with 
    real-time status synchronization.
- 🛡️ RBAC: Role-Based Access Control (Admin vs. Member permissions).

================================================================================
3. TECH STACK
================================================================================
FRONTEND:
- React 18 + Vite (Fast builds & optimized performance)
- Framer Motion (Premium micro-animations)
- Lucide React (Clean, modern iconography)
- Vanilla CSS (Custom design system with Glassmorphism)

BACKEND:
- Node.js + Express.js (Scalable REST API)
- Prisma ORM (Type-safe database access)
- PostgreSQL (Robust relational data management)
- JSON Web Tokens (JWT) for secure session handling
